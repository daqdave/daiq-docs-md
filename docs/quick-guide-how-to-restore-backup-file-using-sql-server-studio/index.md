# How to Restore a Backup File Using SQL Server Studio

![How Restore Backup File Using SQL Server Studio image 01](./images/quick-guide-how-to-restore-backup-file-using-sql-server-studio_01.png)

1. Use *SQL Server Management Studio* to restore the backup file.

![How Restore Backup File Using SQL Server Studio image 02](./images/quick-guide-how-to-restore-backup-file-using-sql-server-studio_02.png)

![How Restore Backup File Using SQL Server Studio image 03](./images/quick-guide-how-to-restore-backup-file-using-sql-server-studio_03.png)

2. Select the *Device* option and browse to the file.

![How Restore Backup File Using SQL Server Studio image 04](./images/quick-guide-how-to-restore-backup-file-using-sql-server-studio_04.png)

3. Click *Add* to select the file path.

![How Restore Backup File Using SQL Server Studio image 05](./images/quick-guide-how-to-restore-backup-file-using-sql-server-studio_05.png)

![How Restore Backup File Using SQL Server Studio image 06](./images/quick-guide-how-to-restore-backup-file-using-sql-server-studio_06.png)

4. Browse folders and select the backup file provided.

![How Restore Backup File Using SQL Server Studio image 07](./images/quick-guide-how-to-restore-backup-file-using-sql-server-studio_07.png)

![How Restore Backup File Using SQL Server Studio image 08](./images/quick-guide-how-to-restore-backup-file-using-sql-server-studio_08.png)

5. Make sure that the database *DAQC5* is selected for restore and click *OK*.

![How Restore Backup File Using SQL Server Studio image 09](./images/quick-guide-how-to-restore-backup-file-using-sql-server-studio_09.png)

You should see a successful result.

![How Restore Backup File Using SQL Server Studio image 10](./images/quick-guide-how-to-restore-backup-file-using-sql-server-studio_10.png)

![How Restore Backup File Using SQL Server Studio image 11](./images/quick-guide-how-to-restore-backup-file-using-sql-server-studio_11.png)

6. Next,  run *Blade Runner* (Testclient.exe).
You will need to login as shown using your *sa* account.
