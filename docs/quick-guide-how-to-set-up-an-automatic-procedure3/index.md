# How to Set Up an Automatic Procedure

![How to Set Up an Automatic Procedure image 01](./images/quick-guide-how-to-set-up-an-automatic-procedure3_01.png)

Note: in newer versions of *StarWatch SMS*, procedures can be found in the administration panel after
selecting the *Admin* button from the main toolbar.

### Step 1:  Open *Triggers and Actions*, as shown.

![How to Set Up an Automatic Procedure image 02](./images/quick-guide-how-to-set-up-an-automatic-procedure3_02.png)

![How to Set Up an Automatic Procedure image 03](./images/quick-guide-how-to-set-up-an-automatic-procedure3_03.png)

### Step 2:  Click on *Automation* to create a new automatic procedure.

![How to Set Up an Automatic Procedure image 04](./images/quick-guide-how-to-set-up-an-automatic-procedure3_04.png)

### Step 3:  Enter a new name for the new procedure.

![How to Set Up an Automatic Procedure image 05](./images/quick-guide-how-to-set-up-an-automatic-procedure3_05.png)

![How to Set Up an Automatic Procedure image 06](./images/quick-guide-how-to-set-up-an-automatic-procedure3_06.png)

### Step 4:  Select the trigger.

By default, the calendar (scheduled) trigger is shown. Click and select the trigger required to invoke
the required actions. In this example, we will describe the *Device Object State Change*d trigger.

![How to Set Up an Automatic Procedure image 07](./images/quick-guide-how-to-set-up-an-automatic-procedure3_07.png)

![How to Set Up an Automatic Procedure image 08](./images/quick-guide-how-to-set-up-an-automatic-procedure3_08.png)

### Step 5:  Select the panel.

From the list of devices (panels), choose the devices that have the relevant input to monitor (trigger
off).

![How to Set Up an Automatic Procedure image 09](./images/quick-guide-how-to-set-up-an-automatic-procedure3_09.png)

![How to Set Up an Automatic Procedure image 10](./images/quick-guide-how-to-set-up-an-automatic-procedure3_10.png)

### Step 6:  Select the point.

From the list of points, choose the relevant input to trigger from.

![How to Set Up an Automatic Procedure image 11](./images/quick-guide-how-to-set-up-an-automatic-procedure3_11.png)

![How to Set Up an Automatic Procedure image 12](./images/quick-guide-how-to-set-up-an-automatic-procedure3_12.png)

### Step 7:  Select the state.

Choose the input state required to trigger from.

![How to Set Up an Automatic Procedure image 13](./images/quick-guide-how-to-set-up-an-automatic-procedure3_13.png)

![How to Set Up an Automatic Procedure image 14](./images/quick-guide-how-to-set-up-an-automatic-procedure3_14.png)

### Step 8:  Add the actions.

For each action required, select an action from the right side actions list and drag it to the center list
of actions. If there are several actions needed, they can be ordered by dragging.

![How to Set Up an Automatic Procedure image 15](./images/quick-guide-how-to-set-up-an-automatic-procedure3_15.png)

### Step 9:  Select an output command and make it active.

Just like for choosing the relevant device, point, and state, choose the command output needed from
the pull-down lists. When you are ready to start monitoring the input and triggering the action, click on
the *Active* checkbox - in effect, make it “live”.

![How to Set Up an Automatic Procedure image 16](./images/quick-guide-how-to-set-up-an-automatic-procedure3_16.png)

![How to Set Up an Automatic Procedure image 17](./images/quick-guide-how-to-set-up-an-automatic-procedure3_17.png)

Step 10.  Save the procedure.
You must save the new procedure.

![How to Set Up an Automatic Procedure image 18](./images/quick-guide-how-to-set-up-an-automatic-procedure3_18.png)

Step 11.  Adding a schedule or holiday.
It may be required to:
a) Not always have the procedure active (ex. specific holidays)
b) Start the procedure from a specific date in the future
c) Have the procedure only be active on certain days of the week

![How to Set Up an Automatic Procedure image 19](./images/quick-guide-how-to-set-up-an-automatic-procedure3_19.png)

### Step 12:  Select the *Schedule* checkbox.

![How to Set Up an Automatic Procedure image 20](./images/quick-guide-how-to-set-up-an-automatic-procedure3_20.png)

Once the schedule checkbox is selected, you can modify the start date or days of the week when the
procedure should be active.

![How to Set Up an Automatic Procedure image 21](./images/quick-guide-how-to-set-up-an-automatic-procedure3_21.png)

![How to Set Up an Automatic Procedure image 22](./images/quick-guide-how-to-set-up-an-automatic-procedure3_22.png)

### Step 13:  Create holiday exceptions.

In the *Linked Calendar* area, select the *Restrict by times and dates* checkbox and choose the calendar
that is used to control the holiday exceptions. The procedure will be limited by the dates in the
calendar that are blocked out.

![How to Set Up an Automatic Procedure image 23](./images/quick-guide-how-to-set-up-an-automatic-procedure3_23.png)

### Step 14:  Perform a manual test.

To test the actions, simply click *Execute*.

![How to Set Up an Automatic Procedure image 24](./images/quick-guide-how-to-set-up-an-automatic-procedure3_24.png)
