# How to Make and Use RPI Templates

![How to Make and Use RPI Templates image 01](./images/quick-guide-how-to-make-and-use-rpi-templates_01.png)

## Introduction

This is a quick guide to making an RPI template and using the template to create new stations.

![How to Make and Use RPI Templates image 02](./images/quick-guide-how-to-make-and-use-rpi-templates_02.png)

### Step 1 – Create a station and points

Using the *RPI Editor*, make your station, zone, and points. This station, called “199”, has one alarm
zone, 16 alarm points, and some housekeeping status points.

![How to Make and Use RPI Templates image 03](./images/quick-guide-how-to-make-and-use-rpi-templates_03.png)

### Step 2 – Create the template

Create the template by right-clicking on the station and selecting the *Create Template* option from the
drop-down menu.

![How to Make and Use RPI Templates image 04](./images/quick-guide-how-to-make-and-use-rpi-templates_04.png)

![How to Make and Use RPI Templates image 05](./images/quick-guide-how-to-make-and-use-rpi-templates_05.png)

### Step 3 – Review the template

Click on the *Templates* tab to see the available templates, including the new template.

![How to Make and Use RPI Templates image 06](./images/quick-guide-how-to-make-and-use-rpi-templates_06.png)

![How to Make and Use RPI Templates image 07](./images/quick-guide-how-to-make-and-use-rpi-templates_07.png)

### Step 4 – Rename the new template

Right-click on the newly created template and give it a useful name by choosing the menu option
*Rename*.

![How to Make and Use RPI Templates image 08](./images/quick-guide-how-to-make-and-use-rpi-templates_08.png)

![How to Make and Use RPI Templates image 09](./images/quick-guide-how-to-make-and-use-rpi-templates_09.png)

### Step 5 – Use the template

To use the template, go back to the configuration by clicking on the *Configuration* tab. From the top,
right-click to choose the menu option *Add Item from Template* and select the newly created template.
You should now have a new station that can be renamed, as required.

![How to Make and Use RPI Templates image 10](./images/quick-guide-how-to-make-and-use-rpi-templates_10.png)

![How to Make and Use RPI Templates image 11](./images/quick-guide-how-to-make-and-use-rpi-templates_11.png)
