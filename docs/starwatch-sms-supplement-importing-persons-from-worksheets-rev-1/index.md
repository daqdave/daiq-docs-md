# How to Import Persons from an Existing Worksheet

![How to Import Persons from an Existing Worksheet image 01](./images/starwatch-sms-supplement-importing-persons-from-worksheets-rev-1_01.png)

## 1.0 IMPORTING PERSONS FROM AN EXISTING WORKSHEET

When importing persons, the most important aspect to understand is that the system is designed to
import one *Person Type* at a time. This means that you need a separate .CSV file/Excel sheet for each
*Person Type* you will import, and you will run the import process multiple times; once for each *Person*
*Type* you intend to import.

## 1.1 USING THE IMPORT FUNCTION

![How to Import Persons from an Existing Worksheet image 02](./images/starwatch-sms-supplement-importing-persons-from-worksheets-rev-1_02.png)

From the *Operator* toolbar, click on the *Import*
icon .

![How to Import Persons from an Existing Worksheet image 03](./images/starwatch-sms-supplement-importing-persons-from-worksheets-rev-1_03.png)

![How to Import Persons from an Existing Worksheet image 04](./images/starwatch-sms-supplement-importing-persons-from-worksheets-rev-1_04.png)

The *Persons Import* screen will appear.

![How to Import Persons from an Existing Worksheet image 05](./images/starwatch-sms-supplement-importing-persons-from-worksheets-rev-1_05.png)

![How to Import Persons from an Existing Worksheet image 06](./images/starwatch-sms-supplement-importing-persons-from-worksheets-rev-1_06.png)

## 1.2 PERSONS IMPORT SCREEN

The imported file should be saved as a .CSV file type. Under *Import file path****,*** browse to the file and
then fill in the proper settings in the *Person Settings* area. In this example, we chose the *Staff* person
type and *DAQ Employee card* for the card type. For the import method fill, select the proper settings
and hit the *Next* button.

![How to Import Persons from an Existing Worksheet image 07](./images/starwatch-sms-supplement-importing-persons-from-worksheets-rev-1_07.png)

![How to Import Persons from an Existing Worksheet image 08](./images/starwatch-sms-supplement-importing-persons-from-worksheets-rev-1_08.png)

This is the original document that is being imported to *StarWatch SMS*:

![How to Import Persons from an Existing Worksheet image 09](./images/starwatch-sms-supplement-importing-persons-from-worksheets-rev-1_09.png)

After you hit *Next* for *Persons Import*, the next window will load and show the format you have
imported. In this example, we have *First Name*, *Last Name*, *Date of birth*, and *Gender*. It also shows
that for action it will copy what is in the columns and for *Gender* it will copy the first letter to identify.

![How to Import Persons from an Existing Worksheet image 10](./images/starwatch-sms-supplement-importing-persons-from-worksheets-rev-1_10.png)

![How to Import Persons from an Existing Worksheet image 11](./images/starwatch-sms-supplement-importing-persons-from-worksheets-rev-1_11.png)

Hit the *Import* button to begin the process.

![How to Import Persons from an Existing Worksheet image 12](./images/starwatch-sms-supplement-importing-persons-from-worksheets-rev-1_12.png)

Hit *Finish* when it is done.

![How to Import Persons from an Existing Worksheet image 13](./images/starwatch-sms-supplement-importing-persons-from-worksheets-rev-1_13.png)

## 1.3 VIEWING IMPORTED PERSONS

![How to Import Persons from an Existing Worksheet image 14](./images/starwatch-sms-supplement-importing-persons-from-worksheets-rev-1_14.png)

To view the imported persons, go to the *Persons*
icon from the toolbar and select the person
group for the person type you added. In our example, it was the *Staff* person type defined for the *DAQ*
*Employees* person group.

![How to Import Persons from an Existing Worksheet image 15](./images/starwatch-sms-supplement-importing-persons-from-worksheets-rev-1_15.png)

![How to Import Persons from an Existing Worksheet image 16](./images/starwatch-sms-supplement-importing-persons-from-worksheets-rev-1_16.png)

Now you can register the new persons or give them access cards, etc.
