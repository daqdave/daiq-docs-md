# EntroStar Shows Disconnected After BACnet Discovery

## Symptom

EntroStar states that it is disconnected in *Site Browser*, but the panel has been configured, discovered with BACnet, and is pingable. *Find a Panel* sees it, and it has been associated in *Site Planner*. What could be wrong?

## Resolution

In the *Management Console*, go to the *Services* tab, then *Orbit* → *Settings*, and make sure that the *BACnet Discovery Interface* is selected correctly.
